# Roles
Roles are cosmetic or permission-based additions to members which allow them to present themselves
differently and allow them to do actions in different forms.

## Role Structure

| field         | type          |
|---------------|---------------|
| id            | snowflake     |
| guild_id      | snowflake     |
| name          | string        |
| permissions   | permissions   |
| position      | integer       |
